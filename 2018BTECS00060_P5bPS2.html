C++ Program to Perform oCelsius to oFahrenheit Conversion

Celsius to Fahrenheit Calculator
algorithm for conversion from celsius to fahrenheit

This C++ Program performs Celsius to Fahrenheit conversion.
(The program reads the temperature in oCelsius, multiplies the temperature in oCelsius by 9 / 5 and adds 32 to obtain temperature in oFahrenheit.)

Here is source code of the C++ program which performs Celsius to Fahrenheit conversion.

#include <iostream>
using namespace std;
int main()
{
  int fahrenheit, celsius;
  float fahrenheit, celsius;
  cout << "Enter the temperature in Celsius : ";
  cin >> celsius;
  fahrenheit = (celsius * 9.0) / 5.0 + 32;
  cout << "The temperature in Celsius : " << celsius << endl;
  cout << "The temperature in Fahrenheit : " << fahrenheit << endl;
  return 0;
}

Sample input : Enter the temperature in Celsius : 40
Sample output:
  The temperature in Celsius: 40
  The temperature in Fahrenheit : 140

Variables used in the program:

  1. fahrenheit: stores the temperature in fahrenheit
  2. celsius: stores the temperature in celsius

Complexity:

  Time Complexity of the program is O(1)
  Space Complexity of the pragram is O(1)
